Github Battle

Github Battle is a project following a tutorial on the fundamentals of ReactJS by Tyler McGinnis from reacttraining.com. It allows a user to choose two github users and have them "battle" based on their standing on github as well as showing popular repos in different languages.

Made using ReactJS.

Super Miner Battle Farm

Super Miner Battle Farm (SMBF) is the name of a fighting game community based in the south west of England and South Wales, the purpose of this site is to act as a central resource for all things SMBF and to provide information about upcoming fighting game events in the south west.

This project was a great opportunity to continue learning how to work with ReactJS as well as Facebook's Graph API.
